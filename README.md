## Plot
Кирилл, радикальный нигилист и фембой, устал жить в тоталитарной стране ZаРусь, где принижают таких, как он. Он решается стать беженцем.

Ему предстоит сначала уехать из своего родного города Мухосранск, а потом получить новый паспорт и преодолеть кордон.
